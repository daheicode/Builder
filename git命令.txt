git init
	相当于svn的checkout
git add .
	添加自己需要上传的文件
git commit -m "String"
	提交自己刚刚添加的文件  String 注解  （相当于svn的日志）
git remote add origin https://github.com/...
	确定自己提交的文件上传到自己github的哪个仓库上
git push -u origin master
	提交（如果仓库已经有内容 则不需要加-u）

如果自己第一次用git提交文件到github上，则需要创建自己的ssh
命令如下：
	ssh-keygen -t rsa -C "邮箱地址"
	然后一路回车
	最后在自己的C盘下的用户里找到 .ssh文件
	并复制其下的id_rsa.pub文件里的内容。
